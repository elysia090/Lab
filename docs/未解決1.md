	0.	Notation and model

Fix:
	•	a finite nonempty alphabet Sigma,
	•	a metric space (Y,D),
	•	a target pair semantics
Phi : X_infty -> Y,

where

X_infty = { (x,i,j) : x in Sigma*, 1 <= i < j <= |x| }.

For each N >= 2, define the bounded slice

X_N = { (x,i,j) : x in Sigma*, |x| <= N, 1 <= i < j <= |x| }.

A finite exact-monoid summary architecture is a tuple

A = (S,e,otimes,lambda,q,R),

where:
	•	S is a finite state set,
	•	e in S,
	•	otimes : S x S -> S is associative with identity e,
	•	lambda : Sigma -> S,
	•	q : Sigma* -> S is the induced monoid homomorphism,
	•	R : S^5 -> Y is a total readout.

For a legal decomposition

x = u a m b v

with a,b in Sigma and corresponding pair (i,j), define

h_A(x,i,j) = R(q(u), q(a), q(m), q(b), q(v)).

The width of A is |S|.

For eps >= 0, define the bounded-slice approximate width

W_Phi^eps(N)
:=
min { |S| :
A is a finite exact-monoid architecture and
sup_{(x,i,j) in X_N} D(h_A(x,i,j), Phi(x,i,j)) <= eps }.

The exact width is the special case

W_Phi(N) := W_Phi^0(N).

Because exact bounded realizations always exist, W_Phi^eps(N) is well-defined and finite for every N >= 2 and every eps >= 0.

For r in N, define the infimal width-r bounded-slice error

E_Phi,r(N)
:=
inf {
sup_{(x,i,j) in X_N} D(h_A(x,i,j), Phi(x,i,j)) :
A is a finite exact-monoid architecture and |S_A| <= r
}.

Then

W_Phi^eps(N) = min { r in N : E_Phi,r(N) <= eps }.
	1.	Bounded exact realizability is fully solved

Theorem 1 (bounded exact theorem).
Fix N >= 2 and Phi restricted to X_N.

Then:
	1.	there exists a finite exact summary architecture on X_N;
	2.	there is a unique state-minimal one, up to renaming of states;
	3.	it is constructible from extensional data by finite partition refinement;
	4.	after compilation, it supports:
	•	build in O(n log n),
	•	append and point update in O(log n),
	•	pair query in O(log n),
for dynamic strings whose current length is at most N.

Proof.

Define

U_N := Sigma^{<= N} union {bot}.

Define the total bounded product star on U_N by

x star y = xy    if x,y != bot and |xy| <= N,
x star y = bot   otherwise.

Then U_N is finite, star is associative, the empty word is the identity, and bot is absorbing.

Let T_N be the finite set of typed one-hole five-slot templates whose substitution outcomes are either legal elements of X_N or ill-typed objects. For each template C and each w in U_N, define

Obs(C,w) in Y union {ILL}

by:
	•	Obs(C,w) = Phi(C[w]) if substitution is legal in X_N,
	•	Obs(C,w) = ILL otherwise.

Define the total profile

Prof(w) := ( Obs(C,w) )_{C in T_N }.

Define the initial equivalence

w ~0 w’
iff
Prof(w) = Prof(w’).

This records both exact observed Phi-values and exact legality behavior.

For each a in Sigma, define total left and right actions on U_N:

L_a(x) := a star x,
R_a(x) := x star a.

An equivalence E on U_N is a two-sided congruence for star iff it is stable under all L_a and R_a.

Let P_N^* be the coarsest partition of U_N such that:
	•	P_N^* refines ~0,
	•	P_N^* is stable under all L_a and R_a.

Because U_N is finite, P_N^* exists and is unique.

Let

S_N := U_N / P_N^*.

Write [w] for the P_N^*-class of w.

Define q_N : Sigma* -> S_N by

q_N(x) = [x]    if |x| <= N,
q_N(x) = [bot]  if |x| > N.

Define

e_N := [empty],
lambda_N(a) := [a],
[x] otimes_N [y] := [x star y].

This is well-defined because P_N^* is a congruence, and q_N is a monoid homomorphism because its definition agrees with truncation by bot.

Define the readout R_N on realized legal five-tuples by

R_N([u],[a],[m],[b],[v]) := Phi(u,a,m,b,v),

and define it arbitrarily on unrealized or illegal five-tuples. This is well-defined because P_N^* refines the total one-hole observational partition, so legality and legal Phi-values are invariant under one-coordinate replacement, and therefore under successive replacement in all five coordinates.

This yields an exact finite architecture on X_N.

Minimality follows because the kernel partition of any exact architecture on X_N must refine ~0 and must be stable under all L_a and R_a, hence must refine P_N^*. Therefore no exact architecture can use fewer states.

For implementation, store the q-values of substrings in any balanced binary tree or segment tree over the current length-n string, with n <= N. Internal nodes combine child summaries by otimes_N. Then build takes O(n log n), each append or point update takes O(log n), and each pair query takes O(log n) by combining O(log n) interval fragments to recover the five needed summaries.

QED.
	2.	Exact width profile

Definition.
For each N >= 2,

W_Phi(N) := W_Phi^0(N).

Theorem 2 (monotonicity of exact width).
For all N >= 2,

W_Phi(N) <= W_Phi(N+1).

Proof.
Restrict any exact architecture on X_{N+1} to X_N.
QED.

Thus W_Phi(N) is the canonical exact bounded-slice complexity profile.
	3.	Slice-wise width-r approximation families are automatic

Theorem 3 (automatic slice-wise family existence).
For every Phi : X_infty -> Y, there exists a family (A_r)_{r >= 1} such that:
	•	|S_r| <= r,
	•	each q_r is an exact monoid homomorphism,
	•	each A_r admits logarithmic online operations after compilation,
	•	for every fixed N,
sup_{(x,i,j) in X_N} D(h_{A_r}(x,i,j), Phi(x,i,j)) -> 0 as r -> infinity.

In fact, for every fixed N, the error is eventually exactly zero.

Proof.
If r < W_Phi(2), choose any dummy width-r architecture. If r >= W_Phi(2), define

M(r) := max { M >= 2 : W_Phi(M) <= r }.

This is well-defined because W_Phi(2) <= r and W_Phi is monotone. Let A_r be any exact bounded architecture on X_{M(r)} of width <= r.

Fix N. Once r >= max(W_Phi(2), W_Phi(N)), monotonicity gives M(r) >= N, so A_r is exact on X_N. Hence the error on X_N is eventually zero.
QED.

Therefore the broad statement “there exists a slice-wise convergent width-r family” is always true and is not an open problem.
	4.	Global exact existence is exactly uniform boundedness of the exact width profile

Theorem 4 (global exact criterion).
For K in N, the following are equivalent:

(i) there exists a global exact architecture on X_infty with at most K states;

(ii) W_Phi(N) <= K for every N >= 2.

Equivalently,

global exact finite-state realizability
if and only if
sup_N W_Phi(N) < infinity.

When finite, the minimal global exact width equals

sup_N W_Phi(N) = lim_{N -> infinity} W_Phi(N),

because W_Phi is monotone.

Proof.

(i) => (ii):
Restrict the global architecture to X_N.

(ii) => (i):
For each N, choose a width-K exact architecture A_N on X_N. After relabeling state sets onto a common labeled K-state set, there are only finitely many K-state monoid skeletons (S,e,otimes,lambda), because there are only finitely many choices of identity, multiplication table, and symbol map lambda : Sigma -> S. Pass to a subsequence on which the monoid skeleton is constant. Then the induced homomorphism q is also constant on that subsequence, since it is determined by the fixed skeleton.

Now fix any realized global five-state tuple tau in S^5. By definition, there exists at least one legal decomposition d_tau = (u,a,m,b,v) of some finite total length L_tau such that

(q(u), q(a), q(m), q(b), q(v)) = tau.

For every bounded slice X_N with N >= L_tau along the chosen subsequence, the same decomposition is legal and has the same five-state tuple tau. Since each A_N is exact on X_N, its readout on tau must equal Phi(d_tau). Therefore the readout value on tau stabilizes and is forced.

If d_tau and d’_tau are two legal decompositions with the same global five-state tuple tau, then for every N large enough that both lie in X_N, the exact architecture A_N must assign the same readout value to tau, so Phi(d_tau) = Phi(d’_tau). Hence the forced value is independent of the chosen witness d_tau.

Define the global readout R on realized tuples tau by this common value, and define R arbitrarily on unrealized tuples. Then the fixed skeleton together with R gives a global exact width-K architecture.
QED.
	5.	Basic monotonicities for bounded approximation

Theorem 5.
For all N,M >= 2 and eps,eps_1,eps_2 >= 0:

(a) if N <= M, then
W_Phi^eps(N) <= W_Phi^eps(M);

(b) if eps_1 <= eps_2, then
W_Phi^eps_2(N) <= W_Phi^eps_1(N);

(c) if N <= M, then
E_Phi,r(N) <= E_Phi,r(M) for every r;

(d) if r <= s, then
E_Phi,s(N) <= E_Phi,r(N) for every N;

(e) W_Phi^0(N) = W_Phi(N).

Proof.
Each claim follows immediately by restriction in N and by relaxing either the allowed width or the allowed error.
QED.
	6.	Half-separation rigidity

For fixed N, define the finite-slice output separation

Delta_Phi(N)
:=
min { D(y,y’) : y,y’ in Phi(X_N), y != y’ }.

If Phi is constant on X_N, the problem is trivial. Otherwise Phi(X_N) is finite, so Delta_Phi(N) > 0.

Theorem 6 (half-separation rigidity).
For every fixed N and every eps < Delta_Phi(N) / 2,

W_Phi^eps(N) = W_Phi(N).

Equivalently, below half the output separation on the slice, approximation buys no width reduction at all.

Proof.
An exact architecture has zero error, so

W_Phi^eps(N) <= W_Phi(N).

For the reverse inequality, suppose a width-r architecture has uniform error <= eps < Delta_Phi(N) / 2. If two legal decompositions produced the same five-summary tuple but different exact outputs y != y’, then the common readout value would approximate both within eps, so

D(y,y’) <= 2 eps < Delta_Phi(N),

a contradiction.

Hence every realized five-summary tuple determines a unique exact Phi-value. Replacing each realized readout by that exact value yields an exact width-r architecture on X_N. Therefore r >= W_Phi(N), so

W_Phi^eps(N) >= W_Phi(N).

Combining both inequalities gives equality.
QED.

Corollary 6.1.
If inf_N Delta_Phi(N) > 0, then for every error schedule eta(N) -> 0,

W_Phi^{eta(N)}(N) = W_Phi(N)

for all sufficiently large N.

Thus if outputs do not become asymptotically dense, vanishing-error asymptotics collapse back to exactness.
	7.	Fixed-slice approximation is fully solved in the unrestricted class

Definition (r-state monoid skeleton).
An r-state monoid skeleton is a tuple

K = (S,e,otimes,lambda)

such that:
	•	S is finite with |S| <= r,
	•	(S,e,otimes) is a finite monoid,
	•	lambda : Sigma -> S.

Let q_K : Sigma* -> S be the induced monoid homomorphism.

For a legal decomposition x = u a m b v in X_N, define the induced five-state tuple

tau_K(u,a,m,b,v)
:=
(q_K(u), q_K(a), q_K(m), q_K(b), q_K(v)).

For each realized tau in S^5, define the output fiber

F_K(tau)
:=
{
Phi(u,a,m,b,v) :
(u,a,m,b,v) is legal in X_N and
tau_K(u,a,m,b,v) = tau
}.

This is finite and nonempty whenever tau is realized.

For any finite nonempty A that is a subset of Y, define

rad_Y(A)
:=
inf_{c in Y} sup_{y in A} D(c,y).

Define the skeleton cost

cost(K)
:=
max_{tau realized} rad_Y(F_K(tau)).

Theorem 7 (full fixed-slice finite optimization).
For every N >= 2 and every r in N,

E_Phi,r(N) = min { cost(K) : K is an r-state monoid skeleton }.

Hence, for fixed N and fixed r, the unrestricted bounded-slice approximation problem is a finite optimization problem.

Proof.

There are only finitely many labeled r-state monoid skeletons, because there are only finitely many choices of:
	•	a state set S with |S| <= r,
	•	an identity element e in S,
	•	a multiplication table otimes : S x S -> S,
	•	a symbol map lambda : Sigma -> S.

So the right-hand side is a minimum over a finite set of real numbers.

To prove “<=”, fix a skeleton K. For each realized tau and each delta > 0, choose c_tau in Y such that

sup_{y in F_K(tau)} D(c_tau,y)
<=
rad_Y(F_K(tau)) + delta.

Use K as the state-transition part and define the readout by R(tau) = c_tau on realized tuples and arbitrarily on unrealized tuples. Then the resulting architecture has width <= r and uniform error <= cost(K) + delta on X_N. Letting delta -> 0 gives

E_Phi,r(N) <= cost(K).

Since this holds for every K,

E_Phi,r(N) <= min_K cost(K).

To prove “>=”, let A be any width-<=r architecture. Its state-transition part is some r-state monoid skeleton K_A. On every realized fiber F_{K_A}(tau), the architecture uses a single readout value R(tau), so its worst-case error on that fiber is at least

rad_Y(F_{K_A}(tau)).

Taking the maximum over realized tau gives

sup_{(x,i,j) in X_N} D(h_A(x,i,j), Phi(x,i,j)) >= cost(K_A) >= min_K cost(K).

Taking the infimum over all width-<=r architectures A gives

E_Phi,r(N) >= min_K cost(K).

Thus equality holds.
QED.

Corollary 7.1.
For fixed N:
	•	E_Phi,r(N) is determined exactly by finite search over finite monoid skeletons;
	•	W_Phi^eps(N) is determined exactly by finite search as
min { r : E_Phi,r(N) <= eps }.

Corollary 7.2.
A representation theorem collapsing every bounded approximate architecture to the canonical bounded quotient architecture is not needed in order to solve the fixed-N approximation problem. Fixed-slice approximation is already fully reduced to finite search in the unrestricted class.

Corollary 7.3 (attainment criterion).
The infimum E_Phi,r(N) always exists as a real number. It is attained by some width-<=r architecture whenever every finite nonempty subset of Y admits a minimax center in Y, for example when Y is compact. Without such an assumption, exact attainment of the infimum need not hold.
	8.	Global uniform approximation under the exact geometric hypotheses

Definition (finite eps-center property).
Fix eps >= 0. Say that Y has the finite eps-center property if every finite nonempty subset A of Y with rad_Y(A) <= eps admits a point c in Y such that

sup_{y in A} D(c,y) <= eps.

Definition (eps-ball intersection property).
Fix eps >= 0. Say that Y has the eps-ball intersection property if every family of closed eps-balls in Y with the finite intersection property has nonempty intersection.

Every compact metric space satisfies both properties for every eps >= 0.

Theorem 8 (global uniform approximation criterion).
Fix K in N and eps >= 0. Assume Y has the finite eps-center property and the eps-ball intersection property. Then the following are equivalent:

(i) there exists a global width-K architecture A on X_infty such that

sup_{(x,i,j) in X_infty} D(h_A(x,i,j), Phi(x,i,j)) <= eps;

(ii) E_Phi,K(N) <= eps for every N >= 2.

Equivalently,

global uniform width-K eps-approximation
if and only if
sup_N E_Phi,K(N) <= eps.

Proof.

(i) => (ii):
Restrict the global architecture to each X_N.

(ii) => (i):
For each N, Theorem 7 and the finite eps-center property imply that there exists a width-K architecture A_N on X_N with error <= eps. After relabeling state sets onto a common labeled K-state set, pass to a subsequence on which the K-state monoid skeleton is constant, hence the induced homomorphism q is constant on that subsequence.

Fix any realized global five-state tuple tau in S^5. Define its global output fiber

G_tau
:=
{
Phi(u,a,m,b,v) :
(u,a,m,b,v) is a legal decomposition in X_infty and
(q(u), q(a), q(m), q(b), q(v)) = tau
}.

For each y in G_tau, consider the closed eps-ball

B(y,eps) := { z in Y : D(z,y) <= eps }.

We claim that the family { B(y,eps) : y in G_tau } has the finite intersection property. Indeed, take any finite subset y_1,…,y_t of G_tau, realized by legal decompositions d_1,…,d_t. Let L be the maximum total length among these finitely many decompositions. For every slice X_N in the chosen subsequence with N >= L, all d_1,…,d_t lie in X_N and have the same five-state tuple tau. Since A_N has width K and uniform error <= eps on X_N, its single readout value on tau lies within eps of each y_s. Therefore that readout belongs to the intersection

B(y_1,eps) intersection … intersection B(y_t,eps),

so the intersection is nonempty.

By the eps-ball intersection property, the total intersection

intersection_{y in G_tau} B(y,eps)

is nonempty. Choose any point in that intersection and define it to be R(tau). Do this for every realized tau, and define R arbitrarily on unrealized tuples.

Then for every legal decomposition with tuple tau and output y in G_tau, one has R(tau) in B(y,eps), hence D(R(tau),y) <= eps. Therefore the fixed skeleton together with R gives a global width-K architecture whose uniform error is <= eps.
QED.

For eps = 0 this recovers Theorem 4. In the exact case, no geometric assumption on Y is needed.
	9.	The unrestricted asymptotic theory is closed negatively

The previous theorems solve the bounded-slice problems completely. What remains at the asymptotic level in full unrestricted generality is not a positive universal growth law but a negative classification result: beyond monotonicity and finite bounded-slice existence, no sharper universal law can hold for all Phi.

Theorem 9 (universal asymptotic nonclassification).
Let Sigma be any finite nonempty alphabet.

(a) There exists an explicit semantics Phi_const such that

W_Phi_const(N) = 1

for all N >= 2.

(b) There exists an explicit semantics Phi_len such that

N - 1 <= W_Phi_len(N) <= N + 2

for all N >= 2.

If in addition |Sigma| >= 2, then:

(c) there exists an explicit semantics Phi_pref such that

sum_{k=0}^{N-2} |Sigma|^k
<=
W_Phi_pref(N)
<=
1 + sum_{k=0}^{N} |Sigma|^k

for all N >= 2.

Therefore:
	•	for every nonempty finite alphabet, no universal asymptotic lower bound stronger than 1 can hold for all Phi, and no universal asymptotic upper bound sharper than the trivial bounded-slice finite upper bound can hold for all Phi;
	•	if |Sigma| >= 2, then constant, linear, and exponential exact-width growth all occur explicitly.

Proof.

Part (a).
Let Phi_const be identically equal to some fixed y_0 in Y. Then the one-state architecture with constant readout is exact, so W_Phi_const(N) = 1 for all N.

Part (b).
Let Y = N with the discrete metric, and define

Phi_len(x,i,j) := |u|

for the unique legal decomposition x = u a m b v corresponding to (x,i,j).

Lower bound:
Fix any symbol s in Sigma. For each k in {0,1,…,N-2}, choose the legal decomposition

x_k = s^k s s
with
u = s^k, m = empty, v = empty.

Then

Phi_len(x_k,i,j) = k.

If q(s^k) = q(s^l) for k != l in an exact architecture on X_N, then these two legal decompositions induce the same five-state tuple and therefore the same output, contradiction. Hence at least N-1 distinct states are needed, so

W_Phi_len(N) >= N - 1.

Upper bound:
Take state set S = {0,1,…,N,N+1}, identity 0, symbol map lambda(c) = 1 for every c in Sigma, and monoid law

p otimes q := min(N+1, p + q).

Then q(w) = min(N+1, |w|). Define the readout by

R(q(u), q(a), q(m), q(b), q(v)) := q(u).

On X_N, one has |u| <= N-2, so q(u) = |u|. Hence the architecture is exact on X_N, and

W_Phi_len(N) <= N + 2.

Part (c), assuming |Sigma| >= 2.
Let Y = Sigma* with the discrete metric, and define

Phi_pref(x,i,j) := u

for the unique legal decomposition x = u a m b v corresponding to (x,i,j).

Lower bound:
Fix any symbol a in Sigma. For every distinct u,u’ in Sigma^{<= N-2}, choose the legal decompositions

x = u a a,
x’ = u’ a a

with m = empty and v = empty. Then

Phi_pref(x,i,j) = u,
Phi_pref(x’,i’,j’) = u’.

If q(u) = q(u’) in an exact architecture on X_N, then these two legal decompositions induce the same five-state tuple and therefore the same output, contradiction. Hence every u in Sigma^{<= N-2} must have a distinct state, so

W_Phi_pref(N) >= |Sigma^{<= N-2}| = sum_{k=0}^{N-2} |Sigma|^k.

Upper bound:
By Theorem 1, exact width on X_N is always at most |U_N|, and here

|U_N| = 1 + sum_{k=0}^{N} |Sigma|^k.

So

W_Phi_pref(N) <= 1 + sum_{k=0}^{N} |Sigma|^k.

This proves all claims.
QED.

Corollary 9.1.
For every nonempty finite alphabet, the unrestricted general problem “find a nontrivial universal asymptotic law for W_Phi(N) over all Phi” has no positive solution. Any meaningful asymptotic theorem must impose additional structural hypotheses on Phi.

Corollary 9.2.
In the witnesses above, the output separation on every finite slice is 1 under the discrete metric. Therefore, by Theorem 6, for every fixed eps < 1/2 the same constant, linear, and (when |Sigma| >= 2) exponential behaviors persist for W_Phi^eps(N). Hence the same negative conclusion applies to the broad fixed-error and vanishing-error asymptotic questions in full unrestricted generality.
	10.	The unrestricted intensional synthesis problem is closed negatively

The extensional bounded-slice theory is complete. At the program level, however, the broad unrestricted effective synthesis problem has no algorithmic solution.

Theorem 10 (undecidability of the broad intensional problem).
Fix any finite nonempty alphabet Sigma. Let Y = {0,1} with the discrete metric, and fix any eps < 1/2.

Given an index of a total computable program computing Phi : X_infty -> Y, the following decision problems are undecidable:

(a) whether Phi admits a global exact width-1 architecture;

(b) whether Phi admits a global uniform eps-approximate width-1 architecture.

Proof.

Let M be a Turing machine on blank input. Define M^+ to be the machine that first idles for exactly 3 steps and then simulates M on blank input. Define a total computable semantics Phi_M by

Phi_M(x,i,j) = 1    if M^+ halts within |x| steps,
Phi_M(x,i,j) = 0    otherwise.

This is total and computable because, on input (x,i,j), one can simulate M^+ for exactly |x| steps and test whether it halts during that finite simulation.

If M never halts, then M^+ never halts, so Phi_M is identically 0 on X_infty. Hence Phi_M has a global exact width-1 realization, and therefore also a global uniform eps-approximate width-1 realization.

If M halts, then M^+ halts after some time T >= 3. Therefore Phi_M takes both values 0 and 1 on X_infty: it is 0 on all legal pairs drawn from words of length 2, and it is 1 on all legal pairs drawn from words of length at least T. Every width-1 architecture has exactly one state, so its readout domain S^5 is a singleton and its output is constant on all legal decompositions. Such an architecture cannot realize both 0 and 1 exactly. Since Y is discrete and eps < 1/2, no constant output can be within eps of both 0 and 1, so no width-1 architecture can eps-approximate Phi_M uniformly either.

Hence deciding either (a) or (b) would decide whether M halts, which is impossible.
QED.

Corollary 10.1.
In full unrestricted generality, there is no general effective synthesis algorithm from arbitrary total computable descriptions of Phi. Any positive effective synthesis theorem must restrict the input description model.
	11.	Final exact status after all reducible gaps are removed

Fully solved, unconditionally, at the extensional theorem level:
	•	bounded exact realizability on every X_N;
	•	unique minimal exact quotient on every X_N;
	•	constructive finite partition-refinement synthesis on every X_N;
	•	logarithmic online execution after bounded compilation;
	•	exact width profile W_Phi(N), finite and monotone;
	•	automatic existence of slice-wise convergent width-r families;
	•	global exact realizability if and only if sup_N W_Phi(N) < infinity;
	•	approximate width profile W_Phi^eps(N), finite and monotone;
	•	half-separation rigidity;
	•	full fixed-N unrestricted approximation by finite optimization over finite monoid skeletons;
	•	exact characterization of global uniform width-K eps-approximation under the finite eps-center property plus the eps-ball intersection property, hence in particular under compactness of Y.

Fully solved, negatively, in the broad unrestricted generality:
	•	there is no nontrivial universal asymptotic law for W_Phi(N) or W_Phi^eps(N) beyond monotonicity and bounded-slice finiteness;
	•	in particular, constant and linear growth always occur explicitly, and when |Sigma| >= 2, exponential growth also occurs explicitly;
	•	there is no general effective synthesis algorithm from unrestricted total computable program descriptions, because even the width-1 global decision problems are undecidable.

What remains mathematically meaningful, and is genuinely open only after adding extra structure, is the structure-restricted theory:
	•	asymptotic growth laws under semantic restrictions on Phi;
	•	coupled width-length-error tradeoffs under semantic restrictions on Phi;
	•	effective synthesis for restricted intensional models (for example, finite-state transducers, bounded-type circuit classes, or other decidable description classes);
	•	structural invariants of Phi that force bounded, logarithmic, polynomial, or exponential behavior inside such restricted subclasses.

	12.	Shortest correct takeaway

The old broad “approximate family existence” question is always yes.

The bounded exact problem is fully solved.

The fixed-slice approximate problem is also fully solved in the unrestricted class by finite optimization over finite monoid skeletons.

The global approximate problem is solved exactly under the finite eps-center property plus the eps-ball intersection property, and in particular under compactness of Y.

Beyond that, the unrestricted “remaining open problems” do not admit a nontrivial universal positive solution: in full extensional generality, asymptotic growth has no sharper universal law, and in full intensional generality, effective synthesis is undecidable.

So the true remaining frontier is not the unrestricted general theory. It is the structure-restricted theory: what additional semantic assumptions on Phi force useful asymptotic laws or computable synthesis.

Two small caution notes.

First, the negative asymptotic closure is a “no universal positive classification” result, not a proof that every structure-sensitive asymptotic question is exhausted. It only says that unrestricted generality is too broad.

Second, the global approximate criterion is exact under the stated geometric hypotheses. If one weakens those hypotheses, the implication from slice-wise infimal bounds to an actual global width-K approximant can fail because infima need not be attained on finite fibers.
