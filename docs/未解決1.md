CONSOLIDATED THEOREM-LEVEL STATUS
	0.	NOTATION AND MODEL
====================

Fix:
	•	a finite alphabet Sigma,
	•	a metric space (Y, D),
	•	a target pair semantics
Phi : X_infty -> Y,

where

X_infty = { (x,i,j) : x in Sigma*, 1 <= i < j <= |x| }.

For each N >= 2, define the bounded slice

X_N = { (x,i,j) : x in Sigma^{<= N}, 1 <= i < j <= |x| }.

A finite exact-monoid summary architecture is a tuple

A = (S, e, otimes, lambda, q, R),

where:
	•	S is a finite state set,
	•	e in S,
	•	otimes : S x S -> S is associative with identity e,
	•	lambda : Sigma -> S,
	•	q : Sigma* -> S is the induced monoid homomorphism,
	•	R : S^5 -> Y is a readout.

For a legal decomposition
x = u a m b v
with a,b in Sigma,
the induced predictor is

h_A(x,i,j) = R(q(u), q(a), q(m), q(b), q(v)).

The width of A is |S|.

For eps >= 0, define the bounded-slice approximate width

W_Phi^eps(N)
:= min { |S| :
A is a finite exact-monoid architecture and
sup_{(x,i,j) in X_N} D(h_A(x,i,j), Phi(x,i,j)) <= eps }.

The exact width is the special case

W_Phi(N) := W_Phi^0(N).

Because exact bounded realizations always exist (Theorem 1 below),
W_Phi^eps(N) is well-defined and finite for every N and every eps >= 0.

⸻

	1.	MAIN THEOREM: BOUNDED EXACT PROBLEM
======================================

Theorem 1 (bounded exact theorem).
Fix N >= 2 and Phi : X_N -> Y.

Then:
	1.	there exists a finite exact summary architecture on X_N;
	2.	there is a unique state-minimal one, up to renaming of states;
	3.	it is constructible from extensional data by finite partition refinement;
	4.	after compilation, it supports:
	•	build in O(n log n),
	•	append / point update in O(log n),
	•	pair query in O(log n).

Proof skeleton.

Step 1: bounded algebra.
Define

U_N := Sigma^{<= N} union {bot},

and the total bounded product star by

x star y = xy    if x,y != bot and |xy| <= N,
x star y = bot   otherwise.

Then U_N is finite, star is associative, empty is the identity, and bot is absorbing.

Step 2: total one-hole observational partition.
Let T_N be the finite set of typed 1-hole 5-slot templates.
For each template C and each w in U_N, define

Obs(C,w) in Y union {ILL}

by:
	•	Obs(C,w) = Phi(C[w]) if substitution is legal in X_N,
	•	Obs(C,w) = ILL otherwise.

Define the total profile

Prof(w) := ( Obs(C,w) )_{C in T_N },

and define the initial equivalence

w ~0 w’
iff
Prof(w) = Prof(w’).

This records both:
	•	equality of observed Phi-values,
	•	equality of legality behavior.

Step 3: congruence closure.
For each a in Sigma, define

L_a(x) := a star x,
R_a(x) := x star a.

An equivalence E on U_N is a two-sided congruence for star
iff it is stable under all L_a and R_a.

Let P* be the coarsest partition of U_N such that:
	•	P* refines ~0,
	•	P* is stable under all L_a and R_a.

Because U_N is finite, P* exists and is unique.

Step 4: quotient architecture.
Let

S := U_N / P*.

Write [x] for the P*-block of x.
Define

q(x) := [x],
e := [empty],
lambda(a) := [a],
otimes([x],[y]) := [x star y].

This is well-defined because P* is a congruence.

Define

R([u],[a],[m],[b],[v]) := Phi(u,a,m,b,v)

on legal decompositions.
This is well-defined because P* refines total one-hole profiles, including legality.

Step 5: minimality.
The kernel partition of any exact architecture must:
	•	refine ~0,
	•	be stable under all L_a and R_a.

Hence it refines P*, so no exact architecture can use fewer states.

QED.

	2.	EXACT WIDTH PROFILE
======================

Definition.
For each N >= 2,

W_Phi(N) := W_Phi^0(N).

Theorem 2 (monotonicity of exact width).
For all N >= 2,

W_Phi(N) <= W_Phi(N+1).

Proof.
Restrict any exact architecture on X_{N+1} to the smaller slice X_N.
QED.

So W_Phi(N) is the canonical exact bounded-slice complexity profile.

	3.	THE “SLICE-WISE APPROXIMATE FAMILY” PROBLEM IS AUTOMATICALLY TRUE
====================================================================

Consider the family condition:

there exists A_r with
	•	|S_r| <= r,
	•	q_r exact monoid homomorphism,
	•	logarithmic online operations,
	•	and for every fixed N,
sup_{X_N} D(h_r, Phi) -> 0 as r -> infinity.

Theorem 3 (automatic slice-wise family existence).
For every Phi : X_infty -> Y, such a family exists.

In fact, for every fixed N,
the error is eventually exactly zero.

Proof.
For each N, the bounded exact theorem gives a finite exact architecture of width W_Phi(N).
For each r, choose any M(r) <= r with W_Phi(M(r)) <= r, and let A_r be an exact bounded architecture on X_{M(r)}.
Then |S_r| <= r.
Fix N.
Once r >= max(N, W_Phi(N)), one has M(r) >= N, so A_r is exact on X_N.
QED.

Therefore:
the mere existence of a slice-wise convergent width-r family is not an open problem.
It holds for every Phi.

The real nontrivial issue is asymptotic growth of width as N grows.

	4.	GLOBAL EXACT EXISTENCE = UNIFORM BOUNDEDNESS OF EXACT WIDTH
==============================================================

Theorem 4 (global exact compactness criterion).
For K in N, the following are equivalent:

(i) there exists a global exact architecture on X_infty with at most K states;

(ii) W_Phi(N) <= K for every N >= 2.

Equivalently,

global exact finite-state realizability
iff
sup_N W_Phi(N) < infinity.

When finite, the minimal global exact width is

sup_N W_Phi(N)
= lim_{N -> infinity} W_Phi(N),

because W_Phi is monotone.

Proof sketch.
(i) => (ii):
Restrict the global architecture to X_N.

(ii) => (i):
Choose width-K exact architectures on all X_N.
By finiteness of possible K-state monoid tables and symbol maps, pass to a subsequence with a common monoid law.
Then diagonally stabilize q(x) on each x in Sigma*.
Define the limiting global readout using exact consistency on large enough slices.
QED.

So unbounded exact semantic existence is fully characterized by the bounded exact width profile.

	5.	GENERAL BOUNDED APPROXIMATION: WHAT IS UNCONDITIONALLY SOLVED
================================================================

Recall

W_Phi^eps(N)
:= minimum width needed for uniform error <= eps on X_N.

Theorem 5.1 (basic monotonicities).
For all N and eps:

(a) if N <= M, then
W_Phi^eps(N) <= W_Phi^eps(M);

(b) if eps_1 <= eps_2, then
W_Phi^{eps_2}(N) <= W_Phi^{eps_1}(N).

Also,

W_Phi^0(N) = W_Phi(N).

Proof.
(a) Restrict from X_M to X_N.
(b) Allowing larger error cannot increase required width.
QED.

So W_Phi^eps(N) is the correct two-parameter bounded-slice capacity profile.

	6.	HALF-SEPARATION THEOREM: SMALL APPROXIMATION DOES NOT REDUCE WIDTH
=====================================================================

For fixed N, define the finite-slice output separation

Delta_Phi(N)
:= min { D(y,y’) : y,y’ in Phi(X_N), y != y’ }.

If Phi is constant on X_N, the problem is trivial.
Otherwise, Delta_Phi(N) > 0 because Phi(X_N) is finite.

Theorem 6 (half-separation rigidity).
For every fixed N and every eps < Delta_Phi(N)/2,

W_Phi^eps(N) = W_Phi(N).

Equivalently:
below half the output separation on the slice,
approximation buys no width reduction at all.

Proof.
One inequality is immediate:

W_Phi^eps(N) <= W_Phi(N),

since an exact architecture has zero error.

For the reverse inequality, suppose a width-r architecture has uniform error <= eps < Delta_Phi(N)/2.
If two legal decompositions produced the same 5-summary tuple but different outputs y != y’, then one readout value would approximate both within eps, implying

D(y,y’) <= 2 eps < Delta_Phi(N),

contradiction.

Hence every realized 5-summary tuple determines a unique exact Phi-value.
Replace each realized readout by that exact value.
This yields an exact width-r architecture on X_N.
So r >= W_Phi(N).

Therefore
W_Phi^eps(N) >= W_Phi(N).

Combining both inequalities gives equality.
QED.

Corollary 6.1.
If

inf_N Delta_Phi(N) > 0,

then for every error schedule eta(N) -> 0,

W_Phi^{eta(N)}(N) = W_Phi(N)

for all sufficiently large N.

So if outputs do not become asymptotically dense, vanishing-error asymptotics collapse back to exactness.

	7.	CANONICAL BOUNDED QUOTIENT MODEL: FULL FINITE OPTIMIZATION
=============================================================

There is one restricted but natural subclass where bounded approximation is completely reducible to a finite optimization problem.

Definition (canonical bounded quotient compilation class on X_N).
Fix N.
Consider only architectures built by:
	•	taking a finite star-congruence partition P of U_N,
	•	using the quotient state set U_N / P,
	•	using the quotient product induced by star,
	•	and choosing an arbitrary readout on quotient 5-tuples.

This is exactly the finite compilation class naturally produced by the bounded exact construction.

For such a congruence P, define for each legal decomposition

t_P(u,a,m,b,v)
:= ([u]_P, [a]_P, [m]_P, [b]_P, [v]_P).

For each realized 5-tuple tau, define the output fiber

F_P(tau)
:= { Phi(u,a,m,b,v) :
(u,a,m,b,v) is legal in X_N and
t_P(u,a,m,b,v) = tau }.

For any finite nonempty A subset Y, define the radius

rad_Y(A)
:= inf_{c in Y} sup_{y in A} D(c,y).

Define the partition cost

cost(P)
:= max_{tau realized} rad_Y(F_P(tau)).

Theorem 7 (exact finite optimization in the canonical class).
Within the canonical bounded quotient compilation class,
the optimal width-r error on X_N is exactly

min { cost(P) :
P is a star-congruence on U_N with at most r blocks }.

Proof.
Once P is fixed, the summary map is fixed.
The best readout value on each realized fiber is exactly its radius-minimizing center error.
Different fibers are independent.
Then minimize over all admissible P.
QED.

So in this canonical class, bounded approximation on fixed X_N is a finite combinatorial optimization problem.

Important scope note.
This theorem is proved only for the canonical bounded quotient class.
The unrestricted class of all exact-monoid architectures may, in principle, be larger.
A full representation theorem collapsing the unrestricted class to this canonical class is not proved here.

	8.	GLOBAL UNIFORM APPROXIMATION UNDER COMPACT OUTPUT SPACE
==========================================================

For general metric Y, one compactness step in the approximate global passage can fail.
Under compactness of Y, it goes through.

Theorem 8 (global uniform eps-approximate compactness; compact Y).
Assume Y is compact.
Fix K in N and eps >= 0.

Then the following are equivalent:

(i) there exists a global width-K architecture A on X_infty such that

sup_{(x,i,j) in X_infty} D(h_A(x,i,j), Phi(x,i,j)) <= eps;

(ii) W_Phi^eps(N) <= K for every N >= 2.

Equivalently, under compact Y,

global uniform width-K eps-approximation
iff
sup_N W_Phi^eps(N) <= K.

Proof sketch.
(i) => (ii):
Restrict to each X_N.

(ii) => (i):
Choose width-K eps-accurate architectures on all X_N.
Pass to a subsequence with a common K-state monoid law.
Diagonally stabilize q(x) for every x in Sigma*.

For each realized limiting 5-state tuple tau, all associated outputs must lie within eps of some bounded-slice readout on every sufficiently large slice.
Thus the family of closed eps-balls around those outputs has the finite intersection property.
Compactness of Y gives a common intersection point.
Choose that as the limiting readout for tau.

This yields a global width-K architecture with uniform error <= eps.
QED.

For eps = 0, this recovers Theorem 4.

	9.	WHAT IS ACTUALLY OPEN
========================

After the reductions above, the true frontier is much narrower.

9.1 The main unresolved invariant is asymptotic growth of W_Phi^eps(N)

The central object is

W_Phi^eps(N)
as N -> infinity,

with eps fixed or varying.

This includes:
	•	exact growth via eps = 0,
	•	constant-error growth via eps > 0,
	•	vanishing-error growth via eps = eta(N).

The meaningful open questions are now:

Open Problem A (exact growth).
How fast does W_Phi(N) grow with N?
Examples:
	•	bounded?
	•	O(log N)?
	•	polynomial?
	•	exponential?
	•	characterized by structural invariants of Phi?

Open Problem B (approximate growth).
How fast does W_Phi^eps(N) grow with N for fixed eps > 0?

Open Problem C (coupled tradeoff).
Given width budget b(N) and error target eta(N), when does

W_Phi^{eta(N)}(N) <= b(N)

hold for all sufficiently large N?

This is the real nontrivial approximation problem, because width, slice length, and error are now coupled.

9.2 Effective synthesis remains open

The results above are semantic and extensional.
In general it remains open to determine, from natural intensional descriptions of Phi
(program, circuit, oracle, stochastic model, etc.):
	•	whether global finite-state exact realization exists,
	•	whether global bounded-width eps-approximation exists,
	•	how to compute or bound W_Phi^eps(N),
	•	how to synthesize near-optimal architectures effectively.

9.3 One missing representation theorem remains

For fixed N, the canonical bounded quotient model admits a complete finite optimization.
For the unrestricted bounded approximate class, an exact reduction to that same finite normal form would require a representation theorem showing that arbitrary exact-monoid architectures do not outperform the canonical bounded quotient compilation class.

That theorem is not proved here.

	•	global uniform width-K eps-approximation

	•	asymptotic growth laws of W_Phi^eps(N);

	•	effective synthesis from intensional descriptions of Phi;
	•	a full representation theorem reducing unrestricted bounded approximation to the canonical bounded quotient model.

	11.	SHORTEST CORRECT TAKEAWAY
=============================

The old broad “approximate family existence” question is not the real open problem.
It is always yes.

The bounded exact problem is fully solved.
A substantial part of bounded approximation is also solved.

What remains is the asymptotic theory:
the growth of the bounded-slice capacity profile

W_Phi^eps(N)

as N increases, and the effective synthesis problem for natural descriptions of Phi.
